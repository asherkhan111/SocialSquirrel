/***************************Running Application Instructions***************************/

To run the application, simply click on the SocialSquirrel.exe file which should then open the console.

/*****************************How to use this application******************************/


Users submit commands to the application. There are four commands. “posting”, “reading”, etc. are not part of the commands; commands always start with the user’s name.


Posting
Users can publish messages by entering their username followed by the message they wish to post. User's can enter '->' in between the username and the message but this is optional and will be removed when the post is created.

Reading
A user can see the messages they have posted by simply entering their username.

Following
Users can follow other users so that an aggregated list will appear on their wall.

Wall
Each user has a wall which contains all the messages they have posted, along with all the messages the users they are following have posted.


posting: <user name> -> <message>

reading: <user name>

following: <user name> follows <another user>

wall: <user name> wall
